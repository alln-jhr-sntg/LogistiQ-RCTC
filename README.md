# LogistiQ - Web-based Vehicle Reservation and Fleet Management System with Rule-Based Vehicle Recommendation for Remix Construction and Trading Corporation

### How to run
- Download the Zip file of the repository
- Extract the file and put it inside xampp/htdocs/
- Rename the extracted folder to "lvms"
- Turn on Apache in Xampp
- Go to http://localhost/lvms/ or http://localhost/lvms/index.php?url=login in your browser
- Have fun :)
